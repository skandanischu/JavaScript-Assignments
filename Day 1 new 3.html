1.Number

3.String
4.Boolean

There are many operations for numbers, e.g. multiplication *, division /, addition +, subtraction -, and so on.

2.BigInt: BigInt type was recently added to the language to represent integers of arbitrary length.
A BigInt value is created by appending n to the end of an integer

3.Strings: A string in JavaScript must be surrounded by quotes. Double quotes "abc", Single quotes 'abc',& backticks.

4.Boolean: The boolean type has only two values: true and false.
